Alguns Projetos

Barber shop

Aplicativo web para barbearia, com objetivo de divulgar os serviços prestados, permite o utilizador agendar horário, com um painel de gestão para o administrador.
Tecnologias usadas: HTML5, CSS, JavaScript, PHP e banco de dados MySQL.

Churrascômetro

Uma calculadora de churrasco, desenvolvido para uma churrascaria.
Tecnologias: HTML5, CSS3 e JavaScript.

Weather Today.

Aplicação de clima que utiliza uma API para acessar informações meteorológicas em tempo real. Ele permite ao usuário buscar dados atualizados sobre o tempo em diferentes localidades, inclui temperatura e a velocidade do vento. A interface simples facilita a visualização rápida das condições do tempo, proporcionando uma experiência prática e informativa.

Gestão Financeira.

Sistema financeiro que permite gerir as despesas.
Tecnologias: HTML5, CSS3 e JavaScript, React, Firebase.
